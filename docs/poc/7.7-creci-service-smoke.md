**Date:** 2026-05-19

**Status:** Doc-only spec (real execution deferred to staging — requires 2Captcha API key)

## Goal

Validar end-to-end o serviço `creciService.lookup(numero, uf)` consultando:
1. Conselho Nacional (Turnstile, 21 UFs) — 1 hit positivo + 1 negativo
2. CRECI SP (reCAPTCHA Enterprise v3) — 1 hit (provavelmente bloqueado mar/2026)
3. Cache hit on segunda chamada (mesma `numero+uf`) — sem novo captcha solve
4. Negative cache (CRECI inexistente) — TTL 30d evita re-solve

## Pré-requisitos

- Env var `CAPTCHA_2CAPTCHA_API_KEY` configurada (NÃO commitada; vault local)
- Supabase local rodando (`supabase start`)
- Migration 010 aplicada (`supabase db reset` localmente)
- Saldo 2Captcha ≥ $5 (cada Turnstile ~$0.003, reCAPTCHA E3 ~$0.002)

## Procedure (manual, ~15 min)

```bash
# 1. Apply migration localmente
cd "Real State - Moema"
supabase start
supabase db reset

# 2. Smoke script (criado pelo @qa em staging)
cat > /tmp/creci-smoke.ts << 'TS'
import { createClient } from '@supabase/supabase-js'
import { createCreciService } from '@/lib/scrapers/creci'

const supabase = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL!,
  process.env.SUPABASE_SERVICE_ROLE_KEY!,
)
const captchaKey = process.env.CAPTCHA_2CAPTCHA_API_KEY!

const service = createCreciService({ supabase, captchaKey })

// Test 1: Conselho positivo (CRECI valido conhecido em PR)
const r1 = await service.lookup('12345', 'PR')
console.log('Test 1 — Conselho hit:', r1)

// Test 2: Conselho negativo (numero inexistente)
const r2 = await service.lookup('99999999', 'RJ')
console.log('Test 2 — Conselho miss:', r2)

// Test 3: Cache hit (rodar Test 1 de novo)
const r3 = await service.lookup('12345', 'PR')
console.log('Test 3 — Cache hit:', r3, '(should have cached:true)')

// Test 4: CRECI SP (provavelmente fail mar/2026)
const r4 = await service.lookup('123456-F', 'SP')
console.log('Test 4 — CRECISP:', r4)
TS

# 3. Run with tsx
cd app && npx tsx /tmp/creci-smoke.ts
```

## Failure modes documented

- **2Captcha down/sem saldo:** `captcha-budget.ts` circuit-breaker cobre — service retorna null + log
- **Conselho retorna HTML 5xx ou layout mudou:** parser Cheerio retorna null; tests `__fixtures__/conselho-malformed.html` cobrem
- **CRECISP reCAPTCHA E3 score < 0.5:** parser detecta + retorna null com `error_code='recaptcha_block'`
- **Cache locked (UNIQUE constraint race):** `upsert` com `onConflict: 'numero,uf'` resolve
