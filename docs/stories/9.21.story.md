# Story 9.21 — Radar de subprecificação / atratividade comercial

**Epic:** 9 — ACM Paridade Premium  
**Status:** Draft  
**Priority:** Should→Must para casos com anúncio ativo  
**Origem:** análise método (Andrade 132) · veredito ROI v3

## Story

Como consultora, quero um **índice de subprecificação** (e ação recomendada) quando o anúncio/pretendido está abaixo da faixa técnica, para eu não “cortar preço” por impulso e reposicionar narrativa/fotos/distribuição.

## Acceptance Criteria

1. **AC1 — Inputs:** anúncio vs faixa técnica; gap vs Top5 ITBI; gap vs lente terreno (se houver); tempo exposição / nº anúncios (opt-in); flag concorrência.
2. **AC2 — Saída:** `subprecificacao: { nivel: 'fraca'|'moderada'|'forte'|null, deltaPct, acaoRecomendada, narrativa }`.
3. **AC3 — Caso 132-like:** ~18% abaixo da construção → `forte` + ação “não reduzir preço…”.
4. **AC4 — Capa Lite/laudo:** bloco curto se nivel ≠ null.
5. **AC5 — Sem anúncio:** null, sem inventar.
