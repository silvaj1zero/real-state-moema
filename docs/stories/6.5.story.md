# Story 6.5: Integracao com Fluxo de Captacao

**Epic:** 6 — Busca Parametrica On-Demand com Enriquecimento de Contatos
**Status:** Done
**Priority:** Should
**Estimated Complexity:** M

## User Story

Como consultora RE/MAX,
Quero converter resultados de busca parametrica diretamente em leads no meu funil de captacao,
Para que eu nao precise copiar dados manualmente e tenha rastreabilidade completa do canal de aquisicao.

## Acceptance Criteria

- [x] AC1: Botao "Captar Lead" no card de resultado. Abre modal pre-preenchido com dados da listing (nome, telefone, email, edificio). INSERT em leads com origem='fisbo_scraping', etapa_funil='contato'.
- [x] AC2: Conversao batch: checkboxes + "Captar N selecionados". Modal de confirmacao com preview dos 3 primeiros.
- [x] AC3: Match automatico com edificio via fn_match_listing_edificio on-the-fly se matched_edificio_id nao existe.
- [x] AC4: Rastreabilidade: leads.enrichment_data inclui source_search_id, source_listing_id, search_params_snapshot.
- [x] AC5: Evento intelligence_feed tipo='novo_fisbo' ao criar lead de busca parametrica.
- [x] AC6: Prevencao duplicatas: check telefone + edificio_id antes de INSERT. Modal de aviso se duplicata encontrada.
- [x] AC7: Aba "Buscas" no AgentDashboard com metricas: total resultados, novos, FISBO, leads criados por busca.
