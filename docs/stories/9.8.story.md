# Story 9.8 — Guard-rails metodológicos do ACM (anti-auto-referência + faixa)

**Epic:** 9 — ACM Paridade Premium
**Status:** Done (QA self-check Fable — ver Dev Agent Record)
**Origem:** Auditoria multi-agente 03-Jul-2026 (`docs/acm/AUDITORIA-EVOLUCAO-ACM-20260703.md`, Frente 1)
**Contexto de negócio:** mercado-alvo região Sul + Jardins (foco Moema / Vila Olímpia / Brooklin);
caso real de captação: Rua Honduras 629 (proprietária Clarisia, consultora Luciana Borba).

## Story

Como consultora, quero que o mecanismo de ACM rejeite automaticamente comparáveis/âncoras
que sejam o próprio imóvel-alvo (auto-referência circular) e exponha a faixa de valor por
cenário de aderência, para que o laudo seja defensável perante o proprietário — evitando a
recorrência do incidente Honduras 639 (anúncio do próprio alvo quase usado como comparável,
corrigido à mão em 28-Jun; ver `docs/HANDOFF-SESSION-20260628-honduras-639.md`).

## Acceptance Criteria

1. **AC1 — `isSelfReference(target, comp)`** em `app/src/lib/acm/methodology.ts`: detecta
   auto-referência por (a) mesma rua + distância < 50 m; (b) fingerprint raro (área ±2% +
   vagas iguais + preço ±5% do pretendido); (c) mesma rua + fingerprint parcial. Retorna
   motivos rastreáveis.
2. **AC2 — Gate no orquestrador:** `computeLaudo` exclui auto-referências ANTES de qualquer
   cálculo e reporta `autoReferenciasExcluidas` no resultado (transparência, Art. IV).
3. **AC3 — Faixa, não ponto:** `computeLaudo` expõe `faixaSensibilidade` (min/max de
   mercado e fechamento entre os cenários Todos/Top5/Top3) como base de dado para a política
   de headline (decisão comercial pendente com a Luciana — doc de auditoria §6.2).
4. **AC4 — Regressão intacta:** caso Honduras (fixture) permanece byte-idêntico nos números
   do laudo; teste novo cobre o caso 639 (positivo) e a oferta legítima "Honduras s/nº"
   (negativo, 418 m² / R$ 22,5M / 736 m).
5. **AC5 — Scripts one-off endurecidos (refactors rápidos #6/#9 da auditoria):** leitura de
   `reverify-result.json` com erro amigável em `04`/`05`; aviso explícito no cabeçalho de
   `reverify-web.workflow.mjs` de que é pseudocódigo de Workflow (não Node executável).
