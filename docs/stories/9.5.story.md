## Story

Como operador, quero que os comparáveis do ACM passem por re-verificação web automatizada (status confirmado/parcial/não recuperável + links + screenshots) usando o crawler do Epic 7, para que a defensabilidade do laudo não dependa de navegação manual — sem que NENHUMA fonte web vire âncora de valor.

## Contexto

A Fase 2 externa (manual) mostrou: retry ingênuo em 403 gera ~60% de falsos negativos; portais de luxo e Wayback/GeoSampa recuperam anúncios que a busca simples perde. A 9.26 já entregou o contrato de dados (`validarAnuncioVenda`, `confiancaC5` no passaporte, `desagioReal`) — esta story pluga a COLETA automatizada nesse contrato. O crawler existe (Epic 7: wrapper 7.2, proxy residencial 7.12, wiring 7.13).

## Acceptance Criteria

1. **AC1 — Fluxo Fase B:** comando/rota que recebe a pasta do caso (`docs/acm/<slug>`) e re-verifica cada comparável com anúncio associado via crawler Epic 7 (`createPortalCrawler` + proxy 7.12). Saída alimenta o contrato C-5 existente (`confianca: alta/média/baixa + ressalvas[]`) — zero schema novo.
2. **AC2 — Cruzamento, nunca âncora (Art. IV):** fontes web (portais, Atlas atlasdados.com/sp, ITBImap itbimap.com.br, Wayback, GeoSampa) entram SOMENTE como validação/cruzamento; nenhum valor delas altera `computeLaudo`/mediana/headline. Teste de paridade prova zero drift numérico com Fase B ligada/desligada.
3. **AC3 — Anti-bot honesto:** navegador real (Playwright/Adaptive) para Atlas/ITBImap (ambos 403 com fetch simples); retry com backoff + fail-fast declarado; resultado `nao_recuperavel` é registrado como tal, nunca como "não existe" (lição dos 60% falsos negativos).
4. **AC4 — Evidência:** screenshot por comparável verificado, salvo na pasta do caso e referenciado no passaporte (`evidenciaUrl`/path). Minimização LGPD: sem rehost de fotos de terceiros além do screenshot de verificação.
5. **AC5 — Gate de deploy LGPD:** enquanto o waiver da 7.10 estiver ativo (LIA/counsel pendente), o fluxo roda APENAS local/offline sob demanda do operador; NENHUM cron/deploy de scraping PF. Gate verificável no código (flag/env documentada).
6. **AC6 — Suite:** testes do fluxo (mock de crawler) + regressão ACM completa verde; casos congelados (Honduras/113/132/baluarte-400) inalterados.
