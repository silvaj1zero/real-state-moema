- [x] AC1: Pre-workshop — script SQL `scripts/epic7/build-validation-batch.sql` que seleciona 200 anuncios estratificados de `scraped_listings`:

### Decisions

- **Migration numerada 020 (nao 016):** numero 016 ja consumido por `20260514000009_016_epic7_review_status.sql` (Story 7.8). Numeracao sequencial preservada apos 019_cron_timezone_guard.
- **Tabela dedicada vs colunas em `scraped_listings`:** dedicada — o batch e efemero (1 workshop) e a separacao mantem `scraped_listings.review_status` (7.8) isolado do `luciana_decision` (7.9).
- **Sem TanStack Query no hook:** dataset pequeno (200), 1 usuario, sem realtime. `router.refresh()` apos Server Action e suficiente.
- **Swipe gestures (left/right):** deferred para wave B. Keyboard shortcuts S/N/? + tap em botoes 44px cobrem o caso de uso primario (Luciana provavelmente usa tablet/desktop no workshop).
- **TRUNCATE no script SQL:** deliberadamente fora — destruir decisoes ja gravadas seria perigoso. Re-run idempotente via `ON CONFLICT DO NOTHING`.
