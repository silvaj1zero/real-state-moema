**Status:** Ready

**Estimated Complexity:** L (8-13 pts)

## User Story

Como sistema ACM,
Quero uma biblioteca de cálculo pura que reproduza a metodologia do laudo Honduras,
Para que o valor de mercado, a faixa de fechamento e o Top N sejam derivados deterministicamente dos comparáveis — e não apenas média/mediana simples como hoje.

## Contexto

`calculateAcmStats` (em `useAcm.ts` L230) só faz média, mediana e contagens; `tendencia` é stub. O `MATERIAL_DIDATICO` e o `LAUDO` definem a metodologia completa a operacionalizar.

## Acceptance Criteria

- [ ] AC1: Lib pura `app/src/lib/acm/methodology.ts` (sem deps Next.js — testável isolada), funções:
  - `classifyScore(precoM2Construido, suites?, vagas?, areaConstruida?)` → `'AAA'|'AA'|'A'|'B'` pela régua do didático (AAA ≥ R$40k + 4+ suítes/vagas ou área ≥500m²; AA ≥30k + 3+; A 22–25k; B <22k).
  - `adherenceIndex(target, comp)` = `0.50·simÁreaConstr + 0.20·simÁreaTerreno + 0.30·proximidade`, cada termo normalizado 0–1 conforme didático Parte 1.3.
  - `landPriceByLotSize(comparaveis)` → R$/m² de terreno mediano por faixa (<500 / 500–800 / >800 m²) — efeito-escala.
  - `residualLandValue(params)` → valor residual do incorporador (VGV − obra − demolição − comercialização − custo financeiro − margem), conforme Laudo Sec. 8b.
  - `sensitivityScenarios(comparaveis, target)` → 3 recortes (Todos / Top 5 / Top 3) com valor de mercado e de fechamento.
  - `liquidityAdjustment(valorMercado, fatores)` → aplica fatores incrementais (exposição, regularização, Capex, liquidez) → valor de fechamento. **Os fatores são inputs por imóvel definidos pela consultora** (ex.: -7/-5/-3/-4 no caso Honduras, Laudo Sec. 2), não constantes do código.
- [ ] AC2: `rankByAdherence(target, comparaveis)` retorna a lista ordenada com o índice — base do Top 5/Top 3.
- [ ] AC3: Resultado tipado `AcmLaudoComputation` (valor de mercado, faixa de fechamento, Top N, R$/m² mediano, efeito-escala, co-âncora de terreno, deságio medido a partir de `preco_pedido`/`desagio_percent`).
- [ ] AC4: `AcmSummaryCards` e `AcmTable` passam a exibir Score e índice de aderência (substituir/estender cards atuais).
- [ ] AC5: **Testes de regressão travando o caso Honduras** (Roadmap Passo 3.3): com os 23 comparáveis do laudo, reproduzir mediana ≈ R$ 18.264/m², valor de mercado ≈ R$ 12,4M, faixa de fechamento R$ 10,0–10,5M, Top 3 = {Chiaffarelli 86, Bittencourt 101, Torres Homem 399}. Tolerância documentada.
- [ ] AC6: `calculateAcmStats` legado mantido ou migrado sem quebrar `AcmExportMenu`/`AcmScreen.test.tsx`.
