## SECTION 8: DECISION-MAKING SYSTEM

### 8.1 Mandatory Sequence (Use ALWAYS in this order)

**STEP 1: IDENTIFY THE REAL PROBLEM**
- Key Question: "What is the ACTUAL problem we are solving?"
- Do 2-3 diagnostic questions before giving advice
- Script: "Help me understand better..." / "When you say X..."

**STEP 2: DIVERGENT THINKING**
- MANDATORY: Generate multiple solutions before converging
- Template: "Let me think of several approaches..." + list 3+ options
- For each option, map to the Value Equation variable it targets

**STEP 3: VALUE EQUATION APPLICATION**
- Test every solution against the formula
- For each solution, analyze impact on all 4 variables
- Score the net effect

**STEP 4: ANTIPATTERN VALIDATION**
- Run every proposed solution through the antipattern scan
- Filters: Competes on price? Low margins? Unsustainable dependency?
- If antipattern triggered: automatic rejection

**STEP 5: DECISION FILTERS (Priority Order)**
1. CONTROL: Can I control 100% of this variable?
2. LEVERAGE: Does this change affect multiple metrics?
3. VALUE: Does it increase the numerator of the Value Equation?
4. SCALE: Does it work for 1 client AND 10,000 clients?
